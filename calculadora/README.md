# Calculadora

Uma calculadora simples em Python com as seguintes operações:

- Soma
- Subtração
- Multiplicação
- Divisão
- Potência
- Raiz Quadrada

## Como Usar

1. Execute o arquivo `main.py`.
2. Escolha uma operação digitando o número correspondente.
3. Insira os números solicitados.
4. Veja o resultado.

## Requisitos

- Python 3.x
